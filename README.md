# University Projects

A collection of data science and machine learning projects developed during my
university studies. Each project lives in its own folder with a dedicated README
describing the data, methods, and main findings.

The work spans statistical modelling, supervised learning, and unsupervised
analysis, across both R and Python.

## Projects

| Project | Topic | Stack | Type |
|---|---|---|---|
| [innovation-patents-r](./innovation-patents-r) | Predicting national patenting intensity from socio-economic indicators | R, `glmnet` | Regression / regularization |
| [binary-classification-svm-rf](./binary-classification-svm-rf) | Binary classification on imbalanced tabular data | Python, scikit-learn | Supervised classification |
| [singlecell-unsupervised](./singlecell-unsupervised) | Unsupervised analysis of single-cell RNA-seq under hypoxia | Python, scikit-learn, UMAP | Clustering / dimensionality reduction |
| [ames-house-prices](./ames-house-prices) | House price prediction with extensive feature engineering | Python, scikit-learn | Regression |

## Tools

- **Languages:** Python, R
- **Python:** NumPy, pandas, scikit-learn, matplotlib, seaborn, UMAP, scipy
- **R:** glmnet, MASS, dplyr, ggplot2, lmtest

## Notes

- Some projects use datasets that cannot be redistributed (course-provided or
  licensed data). In those cases the data is **not** included; the notebooks are
  provided with outputs and figures pre-rendered so the full analysis is viewable
  without running them, and the data format is documented in each project README.
- Projects are individual unless otherwise stated in their README.
